# Hürtur Otel Web Sitesi

Yatağan, Muğla'da bulunan Hürtur Otel'in resmi web sitesi.

## 🚀 Hızlı Başlangıç

### Gereksinimler
- Node.js (v16 veya üzeri)
- npm

### Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm start

# Production build al
npm run build
```

## 📁 Klasör Yapısı

```
hurtur-otel/
├── public/
│   ├── images/
│   │   ├── logo.png          # Otel logosu
│   │   └── gallery/          # Galeri fotoğrafları
│   │       ├── gallery-1.jpg
│   │       ├── gallery-2.jpg
│   │       └── ...
│   └── index.html
├── src/
│   ├── App.js                # Ana uygulama
│   └── index.js              # Giriş noktası
└── package.json
```

## 🖼️ Fotoğraf Ekleme

Galeri fotoğraflarını `public/images/gallery/` klasörüne ekleyin:
- `gallery-1.jpg` - Otel dış görünüm
- `gallery-2.jpg` - Oda fotoğrafı
- `gallery-3.jpg` - Kahvaltı salonu
- `gallery-4.jpg` - Lobi
- `gallery-5.jpg` - Bahçe
